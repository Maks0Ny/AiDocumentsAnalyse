# AiDocumentsAnalyse

## 📌 Описание проекта

Данный проект представляет собой программную систему для автоматизированного анализа договоров с целью выявления юридических рисков, ошибок и потенциально проблемных формулировок.

Система использует гибридный подход:
- правиловый анализ (на основе юридических шаблонов);
- нейросетевой анализ (на базе моделей NLP).

---

## 🚀 Возможности системы

- 📄 Загрузка договоров (DOCX, TXT, PDF)
- 🧹 Очистка и нормализация текста
- 🧩 Структурный анализ документа
- ⚖️ Выявление юридических рисков:
  - неопределённые сроки
  - неясные условия оплаты
  - слабая ответственность сторон
  - отсутствие условий разрешения споров
- 🤖 Нейросетевой анализ формулировок
- 📊 Гибридное объединение результатов
- 📝 Генерация итогового отчёта (DOCX / TXT)

---

## 🧠 Используемые технологии

- Python 3.10+
- PyTorch
- HuggingFace Transformers
- ruBERT (cointegrated/rubert-tiny2)
- python-docx
- регулярные выражения (re)
